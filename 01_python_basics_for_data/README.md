# 01. Python Basics for Data Analysis

This section covers the fundamental building blocks of Python, applied specifically to business logic and data cleaning scenarios.

## Concepts Covered
* **Loops & Iteration:** Automating calculations across sales datasets (e.g., calculating weekly revenue).
* **List Comprehensions:** Efficiently filtering transaction data based on specific business rules.
* **Conditional Logic:** Building automated classification systems (e.g., categorizing customers by spending tiers).
* **String Cleaning:** Standardizing messy product names and user input for analysis.

## Business Scenarios Applied
1. **Sales Analysis:** Aggregating daily revenue and calculating averages.
2. **Order Filtering:** Identifying high-value transactions from a bulk list.
3. **Customer Segmentation:** Using `if-elif-else` logic to assign loyalty levels.
