# Python for Data Analytics - Learning Repository

Documenting my Python learning journey with a focus on data analytics applications. Every concept is applied to real business scenarios - sales, inventory, customer data-rather than generic textbook examples.

## Topics Covered
* **Variables & Data Types** → Handling sales, regions, and currency conversion
* **String Manipulation** → Cleaning messy product names and categories
* **Lists & Tuples** → Managing customer IDs and transaction sequences
* **Dictionaries & Sets** → Mapping products to prices and finding unique regions
* **Conditionals** → Building business logic for revenue targets
* **Loops & Iteration** → Calculating total revenue and automating reports
* **Functions & Lambda** → Creating reusable discount and tax calculators
* **File Handling** → Loading and parsing JSON/CSV business data
* **Error Handling** → Building robust scripts for data ingestion
* **Iterators & Generators** → Memory-efficient processing of large datasets

## Why This Approach
I come from a manufacturing background. I learn best when code solves actual business problems, which helps me build an "analyst mindset" while mastering syntax.
